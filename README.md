# Hinge 2006 — "Meaningful Connections. Desktop Edition."

A fully-functional Retro-Future Hackathon MVP of the Hinge matchmaking experience, built as if it launched in **2006** instead of today.

This desktop application utilizes Windows XP/Vista design language, Yahoo/MSN Messenger interactions, and Orkut/MySpace scrapbook profile directories while retaining the core purpose of Hinge.

---

## 🛠️ Technology Stack & Execution

- **Frontend:** React 18 + TypeScript + Vite
- **Styling:** Custom CSS Modules with 3D Aqua Gradients and Windows Luna/Aero themes
- **Routing:** React Router v6
- **Database / Backend:** Simulated local storage backend with automatic peer replies and notifications

---

## 🎨 Authentic 2000s Design Language

- **Luna Blue & Aero Silver themes:** Toggleable via the toolbar address bar panel.
- **Skeuomorphic components:** Glossy active-gradient buttons, collapsible folder sidebars, Windows Control Panel tabs, and beveled panels.
- **No Swiping:** Replaced by a searchable directory with paginated listings, simulating network limits of dial-up modems.
- **MSN/Yahoo Messenger Buddy Lists:** Mutual matches appear on a contact list. Double clicking opens a chat window complete with emoticons and **BUZZ!!! (Nudge)** vibration alerts.
- **MatchBot popup:** Hinge's modern AI algorithms (Compatibility Scores, Trust analysis, and Conversation Starters) are displayed inside a classic Windows tray popup helper rather than a chat interface.

## 🚀 Setup & Execution

### 1. Install Dependencies
Ensure you have Node.js installed, then execute:
```bash
npm install
```

### 2. Launch Development Server
```bash
npm run dev
```
The application will automatically boot at `http://localhost:3000`.

### 3. Build Production Target
```bash
npm run build
```
Produces highly optimized, zero-warning production build packages inside the `/dist` directory.
